# weatherServer2
This app serves a weather web page. It is based on the weatherServer project with the following changes:
- The Flask web application framework and Gunicorn web application server are used rather than the Python http.server library
- Data is retreived from an InfluxDB server rather than MariaDB

The project no longer includes the weather data collection applications. Data collection is now perfomed by the tempest project

## Installation
The server requires configuration to set the necessary parameters for accessing the Influx database. A configuration file with the
necessary data can be generated by running the config script and answering the questions.

The installation script will create the run-time environment in /opt/wserver and copy the wserver.ini configuration file to /usr/local/etc
